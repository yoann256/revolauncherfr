# Comment conneter de sWiimotes?

## Introduction
Depuis Revo Launcher v2.5, la version Windows de l'appli permet de connecter des Wiimotes.\
Si vous voulez connecter vos Wiimotes, regarder ce guide fait par l'équibe de Dolphin Emulator: [Comment connecter des Wiimotes à votre ordinateur (En Anglais!)](https://web.archive.org/web/20241227013638/http://www.dolphin-emulator.com/connect-wiimote.html)

## Fait! Maintenant quoi?
Simplement ouvrez le programme! revo Launcher va automatiquement chercher des Wiimotes connectées au démarrage.\
Vous pouvez aussi aller dans les Paramètres -> Wimmotes -> Appairer des Wiimotess

## Comment utiliser la Wiimote?
Vous pouvez l'utiliser de trois façons différentes:
- Utilisant un senseur IR:
    Le plus cool, mais plus dur. Vous devez allumer votre Wii/Wii U (en mode vWii), et mettre la barre senseur sur votre moniteur.
- Utilisant un Nunchuck: Vous pouvez utiliser le joystick du Nunchuck pour bouger lecurseur.
- Utilisant une Wiimote horizontale: Finalement, vous pouvez mettre votre Wiimote horizontalement et utiliser la croix directionelle pour bouger le curseur.
